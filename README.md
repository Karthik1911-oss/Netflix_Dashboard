# 🎬 Netflix Analysis Dashboard

## 📌 Project Overview
This project presents an interactive **Netflix Analysis Dashboard** built using **Power BI**, based on a dataset sourced from **Kaggle (5000+ records)**. The goal is to analyze Netflix content trends, ratings, genres, and distribution across countries.

---

## ❗ Problem Statement
With the rapid growth of streaming platforms like Netflix, it becomes difficult to understand:
- What type of content is most popular?
- Which countries produce the most content?
- How ratings vary across different titles and genres
- The distribution of Movies vs TV Shows

This project aims to solve these challenges by transforming raw data into meaningful insights through visualization.

---

## 🎯 Objectives
- Analyze Netflix content by **type, genre, country, and year**
- Identify **top-rated titles**
- Compare **Movies vs TV Shows distribution**
- Provide a clear **data-driven dashboard for insights**

---

## 📂 Dataset Information
- 📊 Source: Kaggle  
- 📦 Records: ~5000 rows  
- 📌 Features:
  - Title
  - Type (Movie / TV Show)
  - Country
  - Genre
  - Rating
  - Release Year
  - Duration
  - Description
